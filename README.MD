## 如何运行

- 运行根目录为当前目录
- 打开`src/main/Main.java`文件, 导入需要运行的作业代码, 例如`import com.Homeworks.Homework_1;`
- 修改`Main.java`文件中的`main`方法中的调用作业代码, 例如`Homework_1.run();`
- 运行 `cmd` 命令`java ./src/main/Main.java`以运行`Main.java`文件, 查看运行结果
